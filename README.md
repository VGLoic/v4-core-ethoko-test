# Uniswap v4 Core - Ethoko Test

This repository is a fork of the Uniswap v4 Core repository, created to test the integration of Ethoko in the Uniswap v4 Core project.

End to end tests have been added in order to test a basic push and pull of artifacts using Ethoko.

## Running the tests

Install dependencies:

```bash
pnpm install
```

Run the end to end tests:

```bash
pnpm test:e2e
```
